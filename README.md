## Concept

腕時計のセレクトショップ

## Usage

```bash
# パッケージのインストール
yarn install

# JITモードでTailwindCSSを起動します
yarn dev
```

## Skills Used

- Liquid
- TailwindCSS
- jQuery（スクロールアニメーション）
- Swiper（スライドショー）
- Animsition（ページ遷移）
